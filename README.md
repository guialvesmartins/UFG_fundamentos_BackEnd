# UFG - Fundamentos de BackEnd

Repositório contendo as atividades e projetos realizados nas aulas de Fundamentos de BackEnd da Pós-Graduação FullStack da Universidade Federal de Goiás (UFG).

## Descrição

Coleção de exercícios e projetos práticos desenvolvidos durante o curso de Fundamentos de BackEnd. Inclui implementações em Java com foco em conceitos fundamentais de desenvolvimento server-side.

## Tecnologias

Java, Spring Boot, Banco de Dados, REST APIs, Padrões de Arquitetura

## Conteúdo

Diferentes atividades práticas de desenvolvimento backend, desde conceitos básicos até implementações mais complexas com APIs e persistência de dados.
